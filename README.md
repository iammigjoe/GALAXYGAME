# GALAXYGAME
Galaxy est un jeu de réflexe et de coordination développé avec Python et Kivy / Follow the indicated path as accurately and quickly as possible to achieve the highest score. Each level gets faster, creating a challenging and addictive experience.

🇫🇷 Description (Français)

Galaxy est un jeu de réflexe et de coordination développé avec Python et Kivy.
Le principe est simple mais addictif : suivre le chemin lumineux qui s’affiche à l’écran en appuyant rapidement sur les bonnes cases, sans se tromper.
Plus le joueur progresse, plus la vitesse augmente et le rythme s’intensifie, testant les réflexes et la concentration.

Inspiré du concept de Piano Tiles, Galaxy offre une expérience fluide et minimaliste, avec des animations spatiales et une ambiance galactique.

⸻

🚀 Fonctionnalités principales
	•	🪐 Gameplay basé sur le rythme et la rapidité
	•	💫 Interface fluide et animations spatiales
	•	⚡ Difficulté progressive au fil du temps
	•	🎧 Effets sonores dynamiques et visuels lumineux
	•	🧠 Test des réflexes et de la concentration
	•	🪄 Développé avec Python + Kivy

⸻

🛠️ Technologies utilisées
	•	Langage : Python
	•	Framework : Kivy
	•	Bibliothèques : Random, Time, Kivy Animation & UI modules

⸻

🎯 Objectif du jeu

Suivre le chemin indiqué sans commettre d’erreurs et atteindre le score le plus élevé possible.
La difficulté augmente à chaque étape, rendant le jeu à la fois stressant et captivant.

🇬🇧 Description (English)

Galaxy is a reflex and rhythm-based game built with Python and Kivy.
The goal is simple yet challenging: follow the highlighted path by tapping the correct tiles as they appear — one mistake, and the game is over!
As the speed increases, your reaction time and focus are put to the test.

Inspired by Piano Tiles, Galaxy combines a minimalist interface with smooth space-themed animations and an addictive gameplay loop.

⸻

🚀 Key Features
	•	🪐 Fast-paced, rhythm-based gameplay
	•	💫 Smooth UI and galactic animations
	•	⚡ Progressive difficulty and increasing speed
	•	🎧 Dynamic sound and visual feedback
	•	🧠 Tests reflexes and hand-eye coordination
	•	🪄 Built with Python + Kivy

⸻

🛠️ Tech Stack
	•	Language: Python
	•	Framework: Kivy
	•	Libraries: Random, Time, Kivy Animation & UI modules
⸻

🎯 Gameplay Goal

Follow the indicated path as accurately and quickly as possible to achieve the highest score.
Each level gets faster, creating a challenging and addictive experience.
